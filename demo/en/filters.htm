langrev. h1.Filters p.In addition to procedural image generation, direct pixel manipulation has another great utility: Image filters. To apply them, first, we will get a buffer with the original value of the pixels by image_pixels. A filter is nothing more than a function that changes the color of each point to achieve the desired effect. In this example we implement the convolution algorithm which needs to know the neighborhood environment of each point to process the effect image_filter_win. Depending on the values assigned to the convolution matrix, we will obtain different results.
