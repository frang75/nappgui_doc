h1.Filters p.Además de la generación procedimental de imágenes, la manipulación directa de píxeles tiene otra gran utilidad: La aplicación de filtros. Para ello, en primer lugar, obtendremos un búfer con el valor original de los píxeles mediante image_pixels. Un filtro no es más que una función que cambia el color de cada punto para lograr el efecto deseado. En este ejemplo implementamos el algoritmo de convolución el cual necesita conocer el entorno de vecindad de cada punto para procesar el efecto image_filter_win. En función de los valores asignados a la matriz de convolución obtendremos diferentes resultados.
