langrev. h1.TextView p.TextView are views designed to work with rich text blocks richtext, where fonts, sizes and colors can be combined. We can consider them as the basis of a text editor. In Hello TextView! you have an example of use. li.Use textview_create to create a text view. li.Use textview_writef to add text to the view. li.Use textview_printf to add text in the format of printf. li.Use textview_rtf to add content in Microsoft RTF format. li.Use textview_clear to erase all text. img(richtext.png,0,.8).Plain text and rich text. Comparison between a plain text block and a rich text block. h2.Character format p.One of the advantages of rich text over plain text is the ability to combine different character formats within the same paragraph charformat. Changes will be applied to new text added to the control. p.Use textview_family to change the font. p.Use textview_fsize to change the character size. p.Use textview_fstyle to change the style. p.Use textview_color to change the color of the text. p.Use textview_bgcolor to change the background color of the text. img(charformat.png,0,.45).Typical Character Format Controls. Selection of fonts and text attributes. h2.Paragraph format p.You can also set attributes per paragraph parformat. The new line character '\n' is considered the closing or end of the paragraph. p.Use textview_halign to set to paragraph alignment. p.Use textview_lspacing to set line spacing (line spacing). p.Use textview_bfspace to indicate the vertical space before the paragraph. p.Use textview_afspace to indicate the vertical space after the paragraph. img(parformat.png,0,.45).Typical controls for paragraph formatting. Formatting a paragraph of text. h2.Document format p.Finally we have several attributes that affect the entire document or control. p.Use textview_units to set the text units. p.Use textview_pgcolor to set the background color of the control (page). func(textview_create).Create a text view. fret(TextView*).The text view. func(textview_size).Sets the default size of the view. fpar(TextView*,view).The view. fpar(const S2Df,size).The size. fnote.It corresponds to the Natural sizing of the control.
