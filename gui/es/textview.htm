h1.TextView p.Los TextView son vistas diseñadas para trabajar con bloques de texto enriquecido, donde pueden combinarse tipografías, tamaños y colores richtext. Podemos considerarlas como la base de un editor de texto. En ¡Hola TextView! tienes un ejemplo de uso. li.Utiliza textview_create para crear una vista de texto. li.Utiliza textview_writef para añadir texto a la vista. li.Utiliza textview_printf para añadir texto con el formato de printf. li.Utiliza textview_rtf para añadir contenido en formato RTF de Microsoft. li.Utiliza textview_clear para borrar todo el texto. img(richtext.png,0,.8).Texto plano y texto enriquecido. Comparativa entre un bloque de texto plano y otro de texto enriquecido. h2.Formato de carácter p.Una de las ventajas del texto enriquecido con respecto al texto plano, es la posibilidad de combinar diferentes formatos de carácter dentro del mismo párrafo charformat. Los cambios se aplicarán al nuevo texto que se añada al control. p.Utiliza textview_family para cambiar la tipografía. p.Utiliza textview_fsize para cambiar el tamaño del carácter. p.Utiliza textview_fstyle para cambiar el estilo. p.Utiliza textview_color para cambiar el color del texto. p.Utiliza textview_bgcolor para cambiar el color de fondo del texto. img(charformat.png,0,.45).Controles típicos del formato de carácter. Selección de tipografías y atributos de texto. h2.Formato de párrafo p.Se pueden establecer también atributos por párrafo parformat. El carácter nueva línea '\n' se considera cierre o final del párrafo. p.Utiliza textview_halign para establecer a alineación del párrafo. p.Utiliza textview_lspacing para establecer la separación entre líneas (interlineado). p.Utiliza textview_bfspace para indicar el espacio vertical antes del párrafo. p.Utiliza textview_afspace para indicar el espacio vertical después del párrafo. img(parformat.png,0,.45).Controles típicos del formato de párrafo. Formateo de un párrafo de texto. h2.Formato del documento p.Por último disponemos de varios atributos que afectan a la totalidad del documento o del control. p.Utiliza textview_units para establecer la unidades del texto. p.Utiliza textview_pgcolor para establecer el color del fondo del control (página). func(textview_create).Crea una vista de texto. fret(TextView*).La vista de texto. func(textview_size).Establece el tamaño por defecto de la vista. fpar(TextView*,view).La vista. fpar(const S2Df,size).El tamaño. fnote.Corresponde al Dimensionado natural del control.
