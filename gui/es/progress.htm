h1.Progress p.Las barras de progreso son controles pasivos que muestran el tiempo restante para completar una determinada tarea progress. A medida que transcurra el tiempo deberemos actualizar el control. El estado indefinido mostrará una animación sin indicar estado, lo que será útil cuando no podamos determinar el tiempo requerido. En ¡Hola Slider y Progress! tienes un ejemplo de uso. li.Utiliza progress_create para crear una barra de progreso. li.Utiliza progress_undefined para establecer la barra como indefinida. li.Utiliza progress_value para ir actualizando el progreso de la tarea. img(progress.png,0,.5).ProgressBar en Windows, macOS y Linux. Ventana donde aparecen varios controles barras de progreso. func(progress_create).Crea una barra de progreso. fret(Progress*).El progress. func(progress_undefined).Establece la barra de progreso como indefinida. fpar(Progress*, progress).El progress. fpar(const bool_t, running).TRUE para activar la animación. func(progress_value).Establece la posición del progress. fpar(Progress*, progress).El progress. fpar(const real32_t,value).La posición entre 0.0 y 1.0.
