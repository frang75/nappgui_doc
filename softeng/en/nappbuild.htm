langrev. h1.NAppBuild p.NAppBuild is a contint utility that we use to generate NAppGUI on a daily basis nbuild. It is an application per command line, without a graphical interface, created in C and installed on a Raspberry-PI board. Every two hours, the administrator cron Unix launches nbuild which checks if there have been changes in the source code repository. If so, it will launch a new build on all build grid hosts. img(nbuild.svg,480,.45).Nbuild application installed on a Raspberry-Pi working continuously. Continuous integration server installed on a Raspberry-Pi. p.NBuild depends exclusively on the library core, except json, included in inet, as a format for data input/output nbuild_depends. img(nbuild_depends.svg,320,.4).Nbuild dependencies.
