langrev. h1.Geom2D p.We are facing a geometric calculation library in two dimensions. Geom2D allows working with primitives in the real plane: Points, vectors, transformations, curves and surfaces. It offers only mathematical functionality, that is, it does not define any type of representation or drawing operation. It only depends on core library geom2d_depends, so it can be used in both desktop applications and command line utilities. All types and functions are defined in simple (float) and double precision, in addition to being able to make use of C++ Math templates. img(geom2d_depends.svg,300,.4).Dependencies of geom2d. See sdk. Geom2d library dependency tree. p.All geometric elements are based on (x, y) coordinates in the plane. Geom2D does not assume how these coordinates will be interpreted. That will depend on the reference system defined by the application. The most used are the Cartesian and the screen refsystems, although others could be used where appropriate systems2d. img(refsystems.svg,580,.65).Interpretation of the coordinate (3,2) on monitors (left) and on the Cartesian plane (right).
