h1.Contenedores 2D p.Los contenedores 2D o (Bounding boxes) delimitan la zona del plano que ocupan diferentes elementos geométricos bounding_box2d. Son útiles en la detección de colisiones u operaciones de recortado (clipping), que impiden que se dibujen figuras no visibles, mejorando el rendimiento global. img(bounding_box2d.svg,520,.65).Contenedor con varios elementos geométricos. Dibujo de un plano donde aparece un contenedor 2D que integra varias figuras. func(B2D,null).Comprueba si un contendor es nulo (sin dimensión). fret(bool_t).TRUE si es nulo, FALSE si contiene alguna geometría. fpar(const B2D*,box).El contenedor. func(B2D,add).Amplia las dimensiones del contenedor para albergar al punto de entrada. Si el punto ya está dentro de su área, las dimensiones no se modificarán. fpar(B2D*,box).El contenedor. fpar(const V2D*,p).El punto a incluir. func(B2D,addn).Amplia las dimensiones del contenedor para albergar varios puntos. Es equivalente a llamar sucesivas veces al método b2d_addf. fpar(B2D*,box).El contenedor. fpar(const V2D*,p).Vector de puntos a incluir. fpar(const uint32_t,n).Número de puntos. func(B2D,merge).Amplia las dimensiones de dest para contener a src. fpar(B2D*,dest).El contenedor que será ampliada. fpar(const B2D*,src).El contenedor que debe ser añadido. func(B2D,overlap).Comprueba si dos contenedores están solapados. fret(bool_t).TRUE si están solapados, FALSE si son disjuntos. fpar(const B2D*,box1).Primera caja. fpar(const B2D*,box2).Segunda caja. const(B2D,NULL).Representa un contenedor nulo (sin geometría).
