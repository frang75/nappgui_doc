Rectángulos 2D p.Ún rectángulo (o frame) rectangle2d sirve para ubicar elementos en interfaces de usuario u otros sistemas 2D mediante un punto de origen V2D y un tamaño S2D. img(rectangle2d.svg,520,.65).Posicionamiento de elementos mediante rectángulos. Dibujo de un plano se ubican varios elementos rectangulares, a modo de interfaz de usuario. func(R2D,).Crea un rectángulo a partir de sus componentes. fret(R2D).El rectángulo. fpar(const real, x).Coordenada x del origen. fpar(const real, y).Coordenada y del origen. fpar(const real, width).Ancho. fpar(const real, height).Alto. func(R2D,contains).Comprueba si un punto está dentro del rectángulo. fret(bool_t).TRUE si el punto está dentro. fpar(const R2D*, r2d).Rectángulo. fpar(const real, x).Coordenada x del punto. fpar(const real, y).Coordenada y del punto. func(R2D,clip).Comprueba si un rectángulo está totalmente fuera de un viewport. fret(bool_t).TRUE si el rectángulo r2d se puede recortar. fpar(const R2D*, viewport).Rectángulo contenedor. fpar(const R2D*, r2d).Rectángulo a comprobar. fnote.Útil para evitar procesar o dibujar objetos que estén totalmente fuera del área de visión. const(R2D,ZERO).Valor [0,0,0,0].
