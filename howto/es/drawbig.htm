h1.Dibujos con scroll p.La siguiente aplicación muestra como gestionar un área de dibujo muy grande, de la que únicamente una pequeña porción es visible. Representaremos una rejilla de 2000x2000 celdas, utilizando un control View con barras de scroll. Los objetivos que perseguimos con este ejemplo son: li.Optimizar el evento OnDraw para dibujar únicamente el área visible, evitando lanzar comandos innecesarios. li.Dimensionar las barras de scroll con view_content_size. li.Desplazar el área visible mediante view_scroll_to. li.Obtener el área visible con view_viewport. li.Uso del ratón: Poder hacer click sobre una celda o resaltarla cuando el cursor está sobre ella. li.Uso del teclado: Permitir a la vista capturar el foco y mover la celda activa con las teclas [Left], [Right], [Up] y [Down].
