h1.Drawing on an image p.In this example we see how to generate vector graphics in two different contexts using the same drawing code drawimg_win. On the left side we render directly into the window through a View control. On the right side generate an image using different resolutions. To show it we use a ImageView control configured to stretch the image in case it is smaller than the control itself, which makes clear the loss of quality. The source code is in folder /src/howto/drawimg of the SDK distribution. img(drawimg_win.png,0,1).2D Contexts: Window (left), Image (right).
